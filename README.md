# Система хранения данных для онлайн книжного магазина

Отказоустойчивая система хранения данных для онлайн-книжного магазина: нормализованная модель БД, автоматические миграции, 
сидирование, мониторинг, резервное копирование и кластер PostgreSQL с репликацией и failover. Настроен мониторинг метрик в Grafana.
Проект полностью автоматизирован, наблюдаем и масштабируем

### Использованные технологии

PostgreSQL · Docker · Flyway · Faker · Prometheus · Grafana · Patroni · SQL · PlantUML

* Спроектирована предметная область и составлена ER-диаграмма (20+ сущностей) в 3НФ.
* Развёрнут PostgreSQL в Docker, реализованы идемпотентные миграции Flyway.
* Настроено автосидирование данных с Faker и параметрами окружения.
* Создана ролевая модель доступа (analytic + пользователи из ANALYST_NAMES).
* Развёрнуты Prometheus и Grafana, настроен сбор метрик и дашборд производительности.
* Реализован нагрузочный сервис для сложных SQL-запросов и проведена оптимизация (индексы, денормализация, партиционирование).
* Настроено резервное копирование по CRON с удержанием последних копий.
* Поднят отказоустойчивый кластер PostgreSQL с Patroni и подтверждён автоматический failover.
